重点是ajax的data处。改为了 data : $(‘#noForm’).serialize()
这次ajax采用了post的传值方式，$(‘#noForm’).serialize()调教了整个表单。本页的代码是一个分页的片段，自己写了一下分页，把第几页，每页多少条数据等数据传回后台，但是之前是get方法，由于有模糊查询，之类的，容易出现乱码错误。所以改成了post方式。自己第一次接触隐藏域，也是第一次认识到“$(‘#noForm’).serialize()”，serialize（）将整个表单与提交，这个是我最注重的。前端我是个渣手，虽然后端也很渣。做个随笔记下来。
