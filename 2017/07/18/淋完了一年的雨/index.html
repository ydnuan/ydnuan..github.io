淋完了一年的雨

发表于 2017-07-18

早晨下起了小雨，让我想起了白居易的那首“渭城朝雨浥轻尘”，忽然才察觉到，他的后两句是“劝君更尽一杯酒，西出阳关无故人”这是一首离别诗啊。内心满是伤感。毕业已经一月有余，“渭城”里的那些古人，也物是人非了。独自一人飘荡在这诺大的京城，何时是归期。去年元宵夜，约在黄昏后。今年元宵夜，不见去年人。

傍晚的雨更大，在地铁了读了读萧红的《呼兰河传》，有感而发，曾经小时候经历过的那些事也不再延续了。多久没有看过晚霞火烧云，多久没在被大雨冲刷后的空气里散步，多久没追着彩虹跑过了。多久没见过傍晚后出来卖豆腐脑的阿郎了，多久。。。长大，剥夺了我们很多东西，只希望接下来的日子，能好好享受，好好去过。小时候不太懂，还以为自己不会长大，永远都只会有爸爸妈妈，爷爷奶奶。可是，家已经不再完整了。开始了亲人的离去，长大，就这么开始了，在这么的一片晨雨中开始，也会在这么的一片晨雨中结束… …
